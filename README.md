# job-salary-lookup---analysis
Excel project analyzing job salaries with LOOKUP formulas and salary buckets for data segmentation.
Job Salary Lookup Analysis (Excel)

 Project Overview
This project uses Microsoft Excel to analyze job salary data. Using LOOKUP formulas, the dataset has been enriched with a new column called salary_year_bucket, which categorizes salaries into different ranges for better analysis.

Dataset Columns
- job_title – Name of the job position  
- job_country – Country where the job is located  
- salary_year_avg – Average yearly salary  
- salary_year_bucket – Salary category (e.g., 0-20k, 20k-50k, 50k-100k, 100k+)

Tools Used
- Microsoft Excel
- LOOKUP formulas (VLOOKUP, HLOOKUP, XLOOKUP)

Key Insights
- Salary segmentation helps identify trends in different job roles and locations  
- Provides a clear overview of salary ranges  
- Enhances Excel analytical skills
